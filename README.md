# Maplestory Trackers

Personal project to track MapleStory progress across a variety of areas.

## Hosted Page

The GitHub Pages default page is `index.html`. It provides a top navigation bar for the available trackers:

- Familiar Badges
- Ride the Lightning v.269

## Familiar Badge Tracker

The Familiar Badge Tracker is available at `familiar-badge.html`.

- Checkbox progress is saved in the browser's local storage.
- Use Export/Import inside the tracker to move progress between browsers or devices.
- `familiar-badge-data.json` and `familiar-badge-cards.csv` are included as source/reference data.

## Ride the Lightning Tracker

The v.269 event tracker is available at `ride-the-lightning.html`.

- Task, quest-starter, and shop progress is saved in the browser's local storage.
- The page includes a weekly timeline, progress tracker, quest starter checklist, and shop calculator.
